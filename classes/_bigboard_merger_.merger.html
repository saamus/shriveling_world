  • merge(): void
    • Defined in src/application/bigBoard/merger.ts:792

    The function merge will

    • retrieve all data from csv files
    • introduce the data into tables:
      • [[cities]],
      • [[population]],

      • transportModeCode,
      • [[transportNetwork]],
